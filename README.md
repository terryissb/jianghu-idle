# 江湖挂机录

> 一个水墨国风武侠挂机游戏，基于纯 HTML/CSS/JavaScript 构建，无需服务器，打开即玩。

## 🎮 游戏简介

**江湖挂机录**是一款时间驱动的武侠修炼挂机游戏。即使你关闭浏览器，角色也会持续修炼，再次打开时自动结算离线期间发生的奇遇、危机与日常事件。

## 🗡️ 核心玩法

- **自动挂机**：每 5~15 秒随机触发事件（奇遇 / 危机 / 日常）
- **离线收益**：关闭页面后自动计算，下次打开自动结算
- **境界突破**：从「凡人」修炼至「天人合一」，共 8 大境界
- **武学收集**：白 / 蓝 / 紫 / 金 四品阶，共 32 门武学
- **自动恢复**：气血与内力自动缓慢回复

## 🏆 武学品阶

| 品阶 | 颜色 | 代表武学 |
|------|------|----------|
| 凡品 | ⚪ | 太祖长拳、庄稼把式 |
| 奇品 | 🔵 | 八卦掌、华山剑法、铁布衫 |
| 绝品 | 🟣 | 降龙十八掌·残篇、六脉神剑·残篇、独孤九剑·残篇 |
| 神品 | 🟡 | 九阴真经、九阳神功、易筋经、葵花宝典、如来神掌 |

## 🎲 事件类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 奇遇 | 意外收获，高成功概率 | 跌落悬崖得秘籍、路遇高人指点 |
| 危机 | 九死一生，高风险高回报 | 仇家追杀、走火入魔、误入禁地 |
| 日常 | 修炼日常，稳定收益 | 打坐修炼、行侠仗义、比武切磋 |

## 💾 本地运行

1. 下载 `index.html` 文件
2. 双击用浏览器打开即可

**存档说明**：游戏进度保存在浏览器 `localStorage` 中，每个设备/浏览器独立存档。
