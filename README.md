# 江湖挂机录

> 一个水墨国风武侠挂机游戏，支持网页版和真正的桌面宠物版。

## 🎮 两种玩法

### 1. 网页版（随时打开）

```
https://terryissb.github.io/jianghu-idle/
```

### 2. 桌面宠物版（真正的桌面悬浮窗口）

```bash
# 进入项目目录
cd jianghu-idle

# 方式 A: 使用已下载的 Electron（推荐）
./start.sh

# 方式 B: 使用 npm 安装 Electron
npm install
npm start
```

## 🗡️ 核心玩法

- **自动挂机**：每 5~15 秒随机触发事件（奇遇 / 危机 / 日常）
- **离线收益**：关闭后自动计算，再次打开自动结算
- **境界突破**：从「凡人」修炼至「天人合一」
- **武学收集**：白 / 蓝 / 紫 / 金 四品阶，共 32 门武学

## 🖥️ 桌面宠物特性

| 特性 | 说明 |
|------|------|
| **悬浮窗口** | 透明无边框，始终置顶，可拖动 |
| **事件气泡** | 实时弹出事件通知 |
| **系统托盘** | macOS 菜单栏图标，右键菜单 |
| **全局快捷键** | `Ctrl + Shift + J` 显示/隐藏 |
| **独立存档** | 与网页版数据互不影响 |
